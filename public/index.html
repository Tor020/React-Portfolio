About

At time of writing in 2017, I am a 26 year-old, self-taught, Web Developer who has been using the Internet since the late 90s. I have always thought the Internet was pretty awesome, but I especially like the practical application it brings to everyone.
The first web application I made and put on github for use is called the KF2calc which helps players calculate their damage in an online video game. Damage calculations are not always displayed anywhere in-game or online. Originally, you would have to look up formulas on the Internet then plug them into a calculator for each weapon. After doing it myself a few times, I thought that it'd be useful to have an application that did this for me. So, I went to building the website.
I launched the website last year and have implemented additional features after communicating with the userbase on social media. Since then, I've monetized the website with Ad Sense and have daily usage confirmed by Google Analytics. I have also launched my own Wordpress website in order to pursue freelance opportunities.
I am really dedicated to mastering Javascript and working to be able to utilize it to its full potential. After I have mastered Front-End Technologies, I'd like to use Javascript on the Back-End and work towards being a Full-Stack Javascript Developer. Eventually, I'd like to use this knowledge in entrepreneurial endeavors.
